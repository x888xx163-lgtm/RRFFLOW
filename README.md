# RFFlow static site v3

Статический сайт RFFlow с расширенной структурой промышленного портала.

## Новое в версии 3

- Умный конфигуратор подбора (`configurator.html`)
- Раздел применений и отдельные страницы задач (`applications/`)
- Отраслевые страницы (`industries/`)
- Центр загрузок (`downloads.html`)
- Проверка прибора по серийному номеру, демо (`serial-search.html`)
- Сравнение приборов (`compare.html`)
- Сервис и поддержка (`service.html`)
- База знаний и SEO-статьи (`knowledge/`)
- Опросные листы (`survey-sheets.html`)
- Улучшенные страницы товаров с вкладками, документами, FAQ и похожими приборами
- Реквизиты ООО «КИС» и карточка поставщика в `docs/kartochka-ooo-kis.pdf`

## Запуск

Откройте `index.html` в браузере или загрузите папку на хостинг.
